Investigadores de la Universidad Autónoma de Madrid (UAM) España, estan buscando la participación en la adquisición de una nueva base de datos a través de una aplicación Android: BehavePassUAM. Esta es una aplicación para adquirir información de nuestra interación normal con dispositivos inteligentes con el objetivo de desarrollar sistemas de reconocimiento biométrico para mejorar la seguridad de los dispositivos móviles.
